# 094 Final Coding Assignment

In this final coding assignment you will create a shopping cart from scratch using HTML, CSS and JavaScript.  

## Create a webpage using the following files: `index.html`, `style.css`, and `index.js`  (Be sure to connect the CSS and JavaScript files into your HTML file in order for them to all work together)
the shopping cart
- should have three items listed on the left side of the screen, each with an "Add" button.
- should display its name and price for each item
- should add the item's name and price to the shopping cart on the right side when the `Add` button is clicked
- should display the total amount
- should incorporate at least 5 CSS styles of your choice (Be creative)
- should have an image for each item
- should contain at least one external link

## Next Steps
**Once you complete the shopping cart please:**
1. Set up a time for a 2 hour Pairing Assessment wit your classroom guide.

2. Explain your code to your classroom guide (`index.html`, `style.css`, and `index.js`)
3. Make 6 live changes to your code with your classroom guide.
